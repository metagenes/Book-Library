## Book Library WebPage Using React

This is a simple book library manager WebPage displaying list of book on Carousel and Cards.



## Dependencies
1. ReactJS
2. 3d-react-carousal
3. Material-ui
4. Axios
5. react-redux
6. redux
7. redux-promise-middleware
8. redux-logger


## Install

    npm install 

## Run the app

    npm start
